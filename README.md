# Ad generator — a Switchboard wrapp

Reads a website's brand with your Claude, then generates ads with your Higgsfield.

Runs on **your own Claude** (and connected tools) through [Switchboard](https://github.com/sameeeeeeep/switchboard) — the operator holds no key, pays for no inference, and sees none of your data.

**Live:** https://adgen.thelastprompt.ai

To use the live features you need the Switchboard extension + the Switchboard app (paired) running locally. Without them the interface still renders; model/storage calls fail honestly. See the [Switchboard repo](https://github.com/sameeeeeeep/switchboard) for setup.
